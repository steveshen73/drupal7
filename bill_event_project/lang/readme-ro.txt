Pentru fiecare limba urmatoarele fisiere trebuiesc încarcate în folderul 
'lang/'- al distributiei LuxCal:

  ai-{limba}.php,        (textul pentru interfata administrator)
  ui-{limba}.php,        (textul pentru interfata utilizator)
  ug-{limba}.php si      (ghidul utilizator ("ajutor")
  ug-layout.png          (imagine de exemplificare comuna tuturor limbilor)


Nota: {limba}, incluzînd acoladele, reprezinta numele limbii.


Dupa încarcarea limbii dorite, limba poate fi selectata din pagina de setari 
a calendarului.
Daca este activata optiunea care permite utilizatorilor sa-si selecteze 
limba, aceasta poate fi selectata din meniul drop-down din bara de navigare.
